> Where can I find the cookies being used on the webpage using the Chrome DevTools?

A: You can find cookies in the DebTools in the "Application" tab, then on the left side under "storage", then "cookies".

> Where can you execute JavaScript in the DevTools?

A: In the "console" tab. 

> How can you modify existing CSS on your webpage using the DevTools?

A: Go to the "elements" tab, and click on the desired html element. In the box on the right, make sure "styles" is selected, and scroll through the various CSS elements, and click on them to edit.

> What happens to the code you have altered through the DevTools when you refresh the page?

A: It gets erased.

> What are the different ways you can add breakpoints to your JavaScript for debugging?

A: You can use Line-of-code, Conditional line-of-code, DOM, XHR, Event listeners, Exceptions, and Functions.
